# Minesweeper

Ported https://github.com/nicolas-maitre/gosu_minesweeper to runa.

## Install

```
$ gem install runa
$ git clone  https://github.com/ongaeshi/minesweeper.git
$ cd minesweeper
$ runa install
$ runa run
```

## Deploy

```
$ runa gen
$ cp .runa/minsweeper.bat path/to/bin
$ minesweeper
```
